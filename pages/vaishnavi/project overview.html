Automated Logistics Scheduling with Shortest Path Optimization

Tasks to be Performed

  • Scheduling Slots: Scheduling truck timings based on available docks.
  • Shortest Path: Optimizing truck routes based on real-time task assignments and shortest paths.

Problem Overview

We manage an automated logistics scheduling system for a large Distribution Centre (DC) handling trucks for multiple warehouses, docks, and storage facilities.

Real-Time Scenario

  • Distribution Centre (DC): The main facility where goods are loaded/unloaded.
  • Warehouses (W1, W2, W3): Storage facilities for goods pick-up or drop-off.
  • Docks (D1, D2, D3): Locations for unloading containers from trucks.
  • Routes: Represented by weighted edges indicating time or distance in minutes.

The trucks need to go through multiple locations to either load or unload goods. We will use Dijkstra's algorithm to determine the shortest path for the trucks and assign the tasks accordingly.

Node Mapping

  • 0: DC (Distribution Centre)
  • 1: W1 (Warehouse 1)
  • 2: W2 (Warehouse 2)
  • 3: W3 (Warehouse 3)
  • 4: D1 (Delivery point 1)
